Depression Detection System

Webcam + Questionnaire-Based Mental Health Screening

Overview

This project uses a webcam and a simple health questionnaire to help detect signs of depression. The system tracks facial expressions for 10 minutes using AI-based emotion detection, alongside user-reported symptoms.

Installation Guide (for Windows)

  1. Install Python 3.7+ from python.org
  2. Install Visual Studio Code from code.visualstudio.com
  3. Open VS Code and install the Python extension
  4. Unzip the downloaded file below and open the folder in VS Code
  5. In the terminal, run:
    pip install opencv-python deepface
  6. Run main.py to start detection
