This system aims to assist elderly and disabled individuals by predicting stress and depression levels using computer vision and AI. It involves software, hardware, and a dedicated website interface.

üß† Detection Methods

  • Facial Emotion Detection: Uses webcam and OpenCV for real-time emotion tracking.
  • Motion Detection: Tracks body movement, stillness, blinking to identify signs of stress or depression.
  • DSM-5 Questionnaire: Interactive form filled by user to score stress/depression risk.
  • Text Input Analysis: Uses NLP to analyze emotional sentiment in free-form text.
  • Report Generation: Combines all methods to generate a downloadable stress report.

üîå Hardware Implementation

  • Raspberry Pi Zero 2 W: Runs lightweight models with USB webcam for real-time tracking.
  • NodeMCU: Sends sensor data or control outputs to display emotion alerts.
  • Webcam: Captures facial expressions and motion data in real-time.
